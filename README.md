# 🔐 CodeAlpha Network Sniffer

## 📌 Project Overview
This project is developed as part of the CodeAlpha Cybersecurity Internship.

The Network Sniffer captures and analyzes live network packets using Python and Scapy. It extracts important information such as source IP, destination IP, protocol type, and packet size.

---

## 🚀 Features
- Capture live network packets
- Display source and destination IP addresses
- Detect protocols (TCP, UDP, ICMP)
- Packet counter
- TCP packet filtering
- Display packet size
- Clean formatted output
- Save packet logs to file

## ▶️ How to Run

### 1. Install Python
Download Python from:
https://www.python.org/

### 2. Install Scapy
```bash
pip install scapy
```

### 3. Run the Program
```bash
python sniffer.py
```
